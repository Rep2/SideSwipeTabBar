# SideSwipeTabBarController

UITabBarController with side swipe.

![Alt Text](https://media.giphy.com/media/l378aiBJqhycXsga4/giphy.gif)

## Usage

Simply inherit from `SideSwipeTabBarController`.

## Installation

### CocoaPods

```ruby
pod 'SideSwipeTabBar'
```
